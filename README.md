# PiconCockpit (PIC)

![Screenshot](pic0.png)

![Screenshot](pic1.png)

## Features
PiconCockpit is a plugin for OA receivers that downloads picons from a picon server.

## Limitations
- PIC is being tested on DM 9xx running OpenVix only
- PIC currently only supports E2-DarkOS skin

## Languages
- english
- german
- italian (by Spaeleus)
